I built a deterministic Daily Reflection Tree that guides users through structured self-reflection without using any AI at runtime.

The system is based on three psychological axes: Locus of Control, Contribution vs Entitlement, and Radius of Concern.

Each axis is encoded as a decision tree where users select fixed options, and the system routes them through predefined nodes.

The key idea is to convert behavioral psychology into a fully traceable, rule-based system so reflection becomes consistent, explainable, and repeatable.

The final output classifies the user’s behavioral pattern and provides a structured reflection summary.

This demonstrates how psychological frameworks can be engineered into deterministic systems instead of relying on LLM-generated responses.
